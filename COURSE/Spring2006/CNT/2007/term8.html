Gradient based search of multiple targets

Presently, search on networks is limited to finding a single target. This can be extended to searching for multiple targets in a network (e.g. searching a document from multiple websites in WWW or searching for multiple resources in peer-to-peer networks etc.)

Task 1: Survey the existing works on decentralized search techniques

Task 2: Devise an algorithm that uses the (local) information about the topology of the n/w to gradually build the set of targets.

Task 3: Evaluate the targets so chosen for one or more particular applications
