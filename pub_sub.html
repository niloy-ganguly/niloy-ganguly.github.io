Alphabetic Bipartite Network

This is a class of evolving bipartite network where one set doesn’t grow with time while the other set grows continuously. For example, if we construct a word – alphabet network, the number of alphabet doesn’t grow with time but the word grows with time. The task has been to find the degree-distribution of the fixed set, and also the degree distribution of the one-mode projection of that set. This class of bipartite network has been  found to be suitable to model various applications ranging from Gene-Codon Networks, Online Social Networks, phoneme-language networks and even Delay Tolerant Network. It has been an amazing journey. Major collaborators -  Animesh Mukherjee, Monojit Choudhury, Fernando Peruani, Tyll Krueger, Sudipta Saha, Saptarshi Ghosh, Abyayananda Maiti

Resilience of Networks

Peer-to-peer networks comprise of nodes which are unreliable in nature, nodes join and leave the networks. The question which one can ask is – is the network robust under such node failures and attacks.  The following set of papers tries to find out the answer from various angles. The papers try to build up analytical framework to answer such question, they consider various sorts of network (correlated, uncorrelated), various sorts of failure and attack and various sorts of methodology (percolation theory, combinatorics etc.) to tackle such problem. Major collaborators - Bivas Mitra, Fernando Peruani, Animesh Srivastava

Growth of Networks

Most of the networks, be it social or technological networks, around us are growing with time. Characterizing the properties of such growing networks has been one of the classical problems of network science. The basic rule which is assumed is preferential attachment that is a new node joins the network by getting connected with popular nodes. In this line of research our simple question has been popular nodes have some constraints, for example in peer-to-peer networks  a node has finite bandwidth while in social networks there is a limit on the number of friends one can have etc.   Major collaborators Bivas Mitra, Saptarshi Ghosh, Joydeep Chandra

Pattern Recognition using Cellular Automata

 This set of papers tackle an important question. Can we build up a machine which will use sparse/regular connection like Cellular Automata and still perform the complex task of pattern recognition. We found a interesting class of Cellular Automata - MACA which divides the state space in such a way that patterns which are close to each other in hamming distance fall under one basin. Major collaborations - Parimal Pal Chaudhuri, Pradipta Maji, Biplab Sikdar.
